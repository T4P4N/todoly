# Todoly
- A simple todo web app (PWA)

## Features
- Localstorage
  - Your todos will stay on page even if reload it
- ~~Strikethrough~~
  - Mark todos as done by tapping the check button
- Delete todos
  - Delete your todos when you are done with them. use X button
